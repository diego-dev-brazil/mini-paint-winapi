# Mini Paint - WinAPI (C)

Projeto desenvolvido em **C puro utilizando a WinAPI**, com o objetivo de estudar:
- Programação gráfica básica
- Manipulação de eventos do Windows
- Gerenciamento manual de desenho (Double Buffer)
- Criação de interface sem frameworks

## 🎨 Funcionalidades
- Menu inicial com botão clicável
- Tela de pintura com mouse
- Botão de cores com menu interativo
- Seleção de cores via interface
- Borracha (cor branca)
- Retorno ao menu com tecla ESC

## 🖱️ Controles
- Clique e arraste para desenhar
- Botão **Cores** para selecionar a cor
- ESC retorna ao menu principal

## 🛠️ Tecnologias
- Linguagem C
- WinAPI (GDI)
- Embarcadero Dev-C++
